#### 3 裴波那契数列
```bash
def fab():
    a, b = 0, 1
    while True:
        yield b
        a, b = b, a+b
n = 0
for i in fab():
    print(i)
    n += 1
    if n == 10:
        break
```

#### 4 素数(质数)算法 -- 自然数中除1之外，只能被1和自身整除
```bash
n = 100
l = []
for i in range(2,n):
    for j in range(2,i):
        if i%j == 0:
           break
    else:
        l.append(i)
print(l)   
```
