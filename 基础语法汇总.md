#### 基础知识和操作
```bash
数字 
  整数：不分long和int,python都是长整型
    进制：0xa, 0o10, 0b10
    bool: 2个值True、False
  浮点数：
  复数：
字符串 
   在字符串前面加r或者R前缀，字符串不做特殊处理
   转义序列：
      \\ \t \r \n \' \"
   缩进：
   续行：
      在行尾使用 \
      如果使用各种括号，认为是一个整体，内部不能使用\
标识符：
    常量
        一旦赋值就不能改变值的标识符
        python中无法定义常量
    字面常量
        一个单独的量， 例如 12、"abc"，放入内存中
    变量
        赋值后，可以改变值的标识符
运算符
    算数运算符
    位运算符
```
