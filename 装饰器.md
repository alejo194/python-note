装饰器是特殊高阶函数---函数作为返回值，函数作为参数
```bash
def logger(fn):
    def wrap(*agrs,**dicts):##参数封装
        # before
        print(agrs,dicts)
        ret = fn(*agrs,**dicts)##参数解构
        # after
        return ret
    return wrap

def add(x,y):
    return x + y
    
foo = logger(add) ## 返回的是一个函数wrap
foo(1,2)

--------------------------
@logger     ## 等价于add=logger(add)
def add(x,y):
    return x + y
    
print(add(4,5))
```
